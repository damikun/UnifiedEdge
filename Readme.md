## Description

Edge Brooker allows you to communicate using different protocols with field devices and bridge between them...

**API**
- Graphql
- Rest
  
**Protocols**
- MQTT (Currently)
- OpcUA (InProgress)
- S7Comm (Planned)

**Environment:**
- Electron multiplatform app
- Self-hosted webApp
